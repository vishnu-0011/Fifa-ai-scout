# FIFA AI Scout

## Project Overview

FIFA AI Scout is an AI-powered agent designed to assist in scouting football players by analyzing their performance and market value data. The agent leverages machine learning techniques to provide insights and recommendations for player scouting, transfers, and team building.

## Dataset

This project uses the [FIFA Player Performance and Market Value Analytics](https://www.kaggle.com/datasets/jayjoshi37/fifa-player-performance-and-market-value-analytics/code) dataset from Kaggle. The dataset contains detailed information on player performance metrics and market values, enabling comprehensive analysis for scouting purposes.

## Features

- Automated player analysis and ranking
- Market value prediction
- Performance-based scouting recommendations
- Data exploration and visualization

## Project Structure

- `main.py`: Entry point for running the AI scout agent
- `src/`: Source code for data processing, agent logic, and utilities
- `data/`: Contains raw and processed datasets
- `models/`: Saved models and checkpoints
- `notebook/`: Jupyter notebooks for data exploration and prototyping

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Download the dataset from Kaggle and place it in `data/raw/`
4. Run the main script: `python main.py`
